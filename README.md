## Bingo Board

Makes a list into a bingo board. Intent is for 2026 bingo resolution board.

### Setup

1. Install python 3.13+

make a list of resolutions, place in bingolist.txt under bingo dir. Must have at least 25 resolutions to fill out bingo board. (No free space)

e.g.
```
build a pond
learn to ice skate
make a pottery closed form
...
```


``` 
cd bingo
python main.py
```
